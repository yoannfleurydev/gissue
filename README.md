# Gissue

Gissue is a software to help you while you're developing awesome softwares.

## Installation

The program is available on [npmjs](https://www.npmjs.com/package/gissue), so you just need npm to install Gissue :

```bash
npm i -g gissue
```

## Development

* Clone the project
* `npm run build`
* `node dist/index.js`
